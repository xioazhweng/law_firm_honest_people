# Юридицеская фирма "Честные люди"

### Build database
```bash
./scripts/reset_db.sh 
```

### Структура проекты
```
my_project/
│
├── README.md                # Описание проекта
├── db/
│   ├── create_db.sql        # SQL для создания базы и схемы (tables, constraints)
│   ├── drop_db.sql          # SQL для удаления/очистки базы (в разработке)
│   └── init_db.py           # Python-скрипт для автоматизации создания базы через psycopg2 (в разработке)
│
├── data_generation/
│   ├── gen_db_.py     # Скрипт на Python с Faker для заполнения таблиц
│
├── queries/
│   ├── 1.sql  
│   ├── 2.sql  
│   ├── 3.sql
│   ├── 4.sql
│   ├── 5.sql
│
└── scripts/
    ├── reset_db.sh          # Скрипт bash для автоматического пересоздания базы
```

### Запросы

1. Получить статистику по пользователям. Отчет представить в виде:
ФИО пользователя, id, дата регистрации, число договоров, общее число услуг, среднее число услуг по договору, дата последнего договора, название самой популярной услуги, список счетов пользователя через запятую.  

2. Получить статистику по работе сотрудников в разрезе по месяцам прошлого года. Отчет представить в следующем виде:
Для каждого сотрудника по одной строке отчета, с 24 столбцами – по 2 столбца на месяц. В первом столбце указывается число договоров в этом месяце, которые оформил сотрудник, во втором – процент изменений относительно прошлого месяца.  

3. Получить информацию о клиентах-должниках. Отчет представить в виде:
ФИО пользователя, число договоров, сумма всех договоров с учетом услуг, сумма всех платежей клиента, дата последнего договора, дата последнего платежа, р/с с которого осуществлялся последний платеж, сумма задолженности.  

4. Получить рейтинг услуг по популярности. Отчет представить в виде:
Название услуги; кол-во договоров, где эта услуга применяется (сортировать по этому полю); среднее число раз включения услуги в договор без учета договоров, где услуга не применялась (значение будет больше или равно 1); текущая стоимость услуги по самому дорогому прайс-листу; текущая стоимость услуги по самому дешевому прайс-листу.  

5. Получить информацию о сумме выплат сотрудникам за текущий год. Отчет представить в виде:
ФИО сотрудника; количество авансов и з/п, которые должны были быть выплачены за текущий год (сотрудник мог быть принять на работу в середине года); сумма всех платежей, которые были выплачены; текущая задолженность перед сотрудником; дата самой старой запланированной выплаты, по которой выплат не было или было выплачено не полностью.
